# Sample

Randomly sample sites from a VCF

```
sample

Randomly sample a VCF

Usage:
  sample [options] vcf

Arguments:
  vcf              Variant file

Options:
  --bed=BED                  A set of bed regions to restrict sampling to
  -n, --sites=SITES          Number of sites to sample (default: 10)
  -h, --help                 Show this help
```
